# API Testing Framework using PyTest

A Python-based API automation testing framework built using **PyTest**, **Requests**, and **GitHub Actions CI/CD**.  
This project demonstrates how to design a scalable and maintainable API testing framework with automated test execution and reporting.

---

## 🚀 Features

- API Automation using **PyTest**
- REST API testing using **Requests library**
- HTML Test Reporting
- Test case structuring with reusable fixtures
- CI/CD integration with **GitHub Actions**
- Modular project structure
- Easy to extend and maintain

---

## 🛠️ Tech Stack

- Python
- PyTest
- Requests
- PyTest HTML Reports
- GitHub Actions (CI/CD)
- Allure Reporting (optional)
